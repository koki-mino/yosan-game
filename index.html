🏙 Ashikaga City Budget Game

シティマスター

あなたはある市の市長です。令和7年度の一般会計当初予算 600億円を、 このゲームでは分かりやすく 100コイン にして考えます。

🎮 ゲームの目的

  • 市長になって、毎年の予算配分を考える。
  • 8つの分野に100コインを分ける。
  • 市民満足度・将来の安心度・財政の健全さの バランスを考える。

📊 8つの分野

  • 教育・子ども / 福祉・高齢者 / 医療・衛生 / 防災・安全
  • 道路・インフラ / まちのにぎわい・産業
  • 行政運営 / 借金返済・貯金

⏱ 1年の流れ

  1. 8つのスライダーで、100コインになるように動かす。
  2. 「この予算で1年すすむ」を押す。
  3. 今年の出来事と、市民の声が表示される。
  4. 3つのメーターの変化を確認する。

💡 気をつけたいこと

  • どこか1つに入れすぎると、ほかが下がることもある。
  • 教育や防災は将来の安心に効きやすい。
  • 借金返済・貯金を増やすと財政はよくなるが、今使えるお金は減る。

ゲームは10年分の予算を決めたあと、 3つのメーターをもとに結果が表示されます。
